# Solidworks Introduction

## 캐드 종류
	
- 2D : 오토캐드
- 3D : 솔리드웍스(다쏘), 솔리에지(경쟁사, 비슷함), 인벤터(오토데스크), 라이노(수치가 중요하지 않을때 곡면설계가 좋다고 한다)
- 3D : 카티아(자동차항공기), 프로이(자동화), UG(금형) - 억대 가격

## 솔리드웍스 단축키

- F : Zoom to fit
- Ctrl+1~7 : 앞,뒤,좌,우,위,아래,ISO
- Mouse Middle : 회전
- Ctrl + Mouse Middle : Move
- Enter : 마지막 커맨드 다시 실행
- Smart Dimension에서 윈의 끝을 찍을 때는 Shift를 누른다. 안그럼 중심이 선택된다.
